Get information with Class Functions

Types of Class Functions

  • client.channels.cache.get('ID'): Finds a specific channel by its id. Returns an object of the GuildChannel class or undefined if not found.
  • client.emoji.cache.get('ID'): Gets an emoji customized by its id. Returns an object of class GuildEmoji or undefined if not found.
  • client.guilds.cache.get('ID'): Gets a specific server by its id. Returns an object of class Guild or undefined if not found.
  • client.members.cache.get('ID'): Gets a specific member by its id. Returns an object of class GuildMember or undefined if not found.
  • client.messages.cache.get('ID'): Gets a specific message by its id. Returns an object of class Message or undefined if not found.
  • client.presences.cache.get('ID'): Gets the presence of a user by their id. Returns an object of class Presence or undefined if not found.
  • client.roles.cache.get('ID'): Gets a specific role by its id. Returns an object of class Role or undefined if not found.
  • client.users.cache.get('ID'): Gets a specific user by their id. Returns an object of class User or undefined if not found.
  • client.voiceStates.cache.get('ID'): Gets the voice state of a member by its id. Returns an object of the VoiceState class or undefined if not found.
