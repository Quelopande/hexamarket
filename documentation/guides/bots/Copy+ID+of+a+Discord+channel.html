How to copy a Discord channel ID

Steps to activate Developer mode:

  1. Below next to your account username you will see the settings section. This symbol:
  2. In the settings section you will see the "Application settings" section.
  3. In the "Application settings" section, go to the "Advanced" section.
  4. There activate the developer mode

How to copy the ID on computers:

  1. Above the channel list square right click.
  2. There you will see the option "Copy channel ID".

How to copy the ID on mobile devices:

  1. Hold down the square in the channel list. As if you were going to edit a channel.
  2. There you will see the option "Copy channel ID".
