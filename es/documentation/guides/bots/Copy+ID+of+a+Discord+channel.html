Como se copiar una ID de un canal de Discord

    Pasos para activar el modo Desarrollador:

  1. Abajo al lado el nombre de usuario de su cuenta verá el apartado ajustes. Este símbolo:
  2. En la sección ajustes verá el apartado "Ajustes de aplicación".
  3. En el apartado "Ajustes de la aplicación" diríjase a el apartado "Avazado".
  4. Ahí activa el modo desarrollador

    Como copiar la ID en ordenadores:

  1. Encima de el cuadrado de la lista de canales haz clic derecho.
  2. Ahí te saldrá la opción "Copiar ID del canal".

    Como copiar la ID en dispositivos móviles:

  1. Mantén pulsado el cuadrado de la lista de los canales. Como si fueras a editar un canal.
  2. Ahí te saldrá la opción "Copiar ID del canal".
